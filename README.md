# Slithify - Playful Cartoon Arcade Game

A beautifully designed **light-aesthetic snake game** with **4 custom pop-culture cartoon themes**, **selectable maze layouts**, animated backgrounds, custom canvas character graphics, synthesized 8-bit audio, and full mobile-responsive layouts!

## 🎮 Themes

| Theme | Snake | Food |
|-------|-------|------|
| 🐱 **Doraemon** | Doraemon face + blue segments | Dora Cake (Dorayaki) |
| 🦑 **Squid Game** | Pink guard mask + green tracksuits (with player numbers!) | Dalgona Candy |
| 😄 **Shinchan** | Thick eyebrows peach head + red/yellow segments | Chocobi Pink Star |
| 🐭 **Tom & Jerry** | Tom cat face + grey/brown segments | Cheese Wedge |

## 🏆 Features

- ✨ Premium **Light Mode** dashboard with white glassmorphic panels
- 🎨 4 cartoon pop-culture themes with **procedural canvas art**
- 🗺️ 5 selectable **maze layouts** (Classic, Box, Cross, Tunnels, Spiral)
- 🎵 Synthesized **8-bit sound effects** via Web Audio API (no audio files needed!)
- ✨ **Particle explosion** effects when eating food
- 🏅 **Top 5 Leaderboard** saved to localStorage
- 📱 **Mobile-friendly** with virtual D-pad and touch swipe controls
- 💻 Vintage LCD handheld screen simulator with glass glare overlay

## 🚀 Play Now

Open `index.html` directly in any modern browser — no server or build step required!

## 🛠️ Tech Stack

- Pure **HTML5**, **CSS3**, and **Vanilla JavaScript**
- **Canvas 2D API** for all game graphics
- **Web Audio API** for synthesized sound generation
- Google Fonts: Orbitron, Fredoka One, Share Tech Mono

## 🕹️ Controls

| Key | Action |
|-----|--------|
| `W` / `↑` | Move Up |
| `S` / `↓` | Move Down |
| `A` / `←` | Move Left |
| `D` / `→` | Move Right |
| `Space` | Pause / Resume |
